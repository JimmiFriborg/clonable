# 1. Concise Product Definition

## Product

Clonable is a local-first AI builder that helps a user turn a product idea into a finished MVP and then continue iterating inside the same real codebase.

## Who it is for

- solo founders
- indie hackers
- small product teams
- technical and semi-technical builders who want AI help without losing control of code, scope, or progress

## Core promise

Clonable does not just generate an app from a prompt. It helps define the actual MVP, breaks work into a usable execution plan, assigns work to specialized agents, applies changes in a real Git workspace, and keeps progress visible and understandable.

## Positioning

Clonable sits between prompt-to-app generators and full project management tools:

- more structured and durable than a one-shot app generator
- more focused and less cluttered than enterprise project software
- more local and controllable than closed hosted builders

## Product principles

- stability over cleverness
- finish the MVP before expanding scope
- visible progress over opaque agent activity
- local repo workflows over abstract sandboxes
- helpful orchestration over chaotic autonomy
- ADHD-approachable clarity over enterprise density

## V1 success statement

A user can enter an idea, define a realistic MVP with the system, see the work broken into phases/features/tasks, run dedicated agents against a real workspace, preview progress locally, and understand what is done, what is next, and what is blocked.
